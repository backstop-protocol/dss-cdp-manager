# B.Protocol

#### Requirements

- [dapp.tools](http://dapp.tools/)

#### Resolve project dependencies

`dapp update`

#### Build project

`dapp --use solc:0.5.16 build`

#### Run tests

`dapp --use solc:0.5.16 test`

#### Run code coverage

`sh coverage.sh`
